# 투표 웹 페이지

투표 기능이 있는 웹 페이지 프로젝트입니다.

## Stack
- javascript
- React
- styled-components(css in js)
- dayjs
- react-toast-notifications
- jest - 테스트

## 기능
- 투표 생성
- 투표 수정, 삭제 (해당 투표 생성자만 가능)
- 투표 참여(한번만 참여 가능)
- 투표 상세보기(투표 정보, 결과)
