# 투표 웹 페이지

투표 기능이 있는 웹 페이지 프로젝트입니다.

## Stack
- React
- 달력
- 상태 관리
- jest - 테스트

## 기능 
- 투표 생성
- 투표 수정, 삭제 (해당 투표 생성자만 가능)
- 투표 참여
- 투표 상세보기(투표 정보, 진행 상황, 결과)
